shadowsocks个人代理解决方案


首先要有一台拥有外网公网IP地址的以CentOS主机为例。



ssh连接服务器

然后输入以下命令依次安装下面两应用

yum install python-pip

pip install shadowsocks

以下命令创建json文件并写入规则

vi /etc/shadowsocks.json

复制下面的东西进入json文件：

这个是可以用的的json文件。

{ "server":"0.0.0.0",
"server_port":8388,
"local_address":"127.0.0.1",
"local_port":1080,
"password":"12345678",
"timeout":300,
"method":"aes-256-cfb",
"fast_open":false
}

输入"i"写入，完成后按EDC键返回 输入":wq"退出

其中"password"修改ssh连接服务器时的root密码，

其他保持默认即可

然后运行：

ssserver -c /etc/shadowsocks.json -d start

开启代理服务

并在服务器后台放行"8388"端口

下载客户端配置服务器ip地址,和服务器密码即可
